# 🔐 Password Strength Checker Web Application

## 📌 Project Description
This project is a web-based password strength checker developed using Flask. It allows users to enter a password and evaluates its strength based on security rules such as length, uppercase letters, lowercase letters, and numbers.

The system helps users create strong and secure passwords to prevent hacking and unauthorized access.

## 🚀 Features
- Real-time password strength checking
- Detects weak, medium, and strong passwords
- Validates password length and complexity
- User-friendly web interface

## 🔐 Security Checks
The system evaluates passwords based on:
- Minimum length (at least 8 characters)
- Presence of uppercase letters
- Presence of lowercase letters
- Presence of numbers

## 🛠️ Technologies Used
- Python (Flask)
- HTML / CSS
- JavaScript (optional for real-time checking)
